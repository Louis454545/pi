# Quickstart

```bash
npm install -g @earendil-works/morgan-agent
morgan
```

Configure authentication with `/login` or the provider environment variable shown by `morgan --help`.

Morgan runs from `$HOME` and continues one global conversation. Put durable personal skills, prompts, extensions, and themes below `~/.morgan/agent/`. Use `/reset` for a fresh conversation and `/export` for a JSONL copy.
